# Sistema de Controle de Máquinas

Projeto desenvolvido em C# como parte dos meus estudos em programação.

O objetivo é praticar lógica de programação, organização de código, criação de métodos, validação de entradas e evolução contínua do projeto.

Este projeto representa meu aprendizado diário e será atualizado conforme eu aprender novos conceitos da linguagem.

## Objetivo

Criar um sistema de console capaz de controlar o fechamento financeiro de máquinas, registrando informações importantes de cada fechamento.

Mais do que desenvolver um sistema funcional, este projeto tem como objetivo consolidar conhecimentos em C# e boas práticas de programação.

## Funcionalidades atuais
✔ Login simples do usuário

✔ Validação de entrada utilizando TryParse

✔ Menu interativo

✔ Consulta de saldo das máquinas

✔ Fechamento de máquinas

✔ Cálculo de lucro

✔ Estrutura organizada em métodos

✔ Tratamento de erros de entrada

## Próximas implementações
- Cadastro de novas máquinas

- Alteração automática do saldo após fechamento

- Histórico de fechamentos
